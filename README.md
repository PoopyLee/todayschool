>使用教程很简单。只需要简单的编译运行即可。
>
>如果不会编译的朋友，请下载本目录下的todayschool.exe文件运行即可。。
>对于有服务器的也下载main文件在你存放该文件的DIR文件下运行一下命令：
>
>先运行
>
>`./main`
>
>进行配置，配置完成之后运行：
>
>`nohup ./main &`
>   
> 即可运行到linux后台
>
